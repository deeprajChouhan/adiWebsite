# Harvest & Co. Rebrand Portal

A production-ready brand portal built with React, Vite, and Material UI.

## Quick start

```bash
npm install
npm run dev
```

Then open `http://localhost:5173`.

## Project structure

```
.
├── index.html
├── package.json
├── vite.config.js
└── src
    ├── App.jsx
    ├── main.jsx
    ├── theme.js
    ├── components
    ├── data
    ├── pages
    └── styles
```
